# 🛠 自定义配置指南

## 🎵 歌曲配置

在 `songs.json` 中添加歌曲：

```json
{
    "id": "19",
    "title": "シャルル",
    "artist": "バルーン",
    "cover": "./jacket/19.png",
    "lyric": "./lyrics/19.xlrc",
    "audio": "音频 URL 或本地路径"
}
```

| 字段 | 说明 |
| ------ | ------ |
| `id` | 歌曲唯一 ID |
| `title` | 歌曲标题 |
| `artist` | 歌手/作者 |
| `cover` | 封面图片路径 |
| `lyric` | 歌词文件路径 |
| `audio` | 音频文件路径 |

> 此处内容可于 [sekai-viewer](https://github.com/Sekai-World/sekai-viewer) 获取

## 📝 歌词格式（.xlrc）

自定义 xlrc 格式，基于 LRC 扩展，支持按角色分词着色和头像显示。

### 基本结构

```xlrc
|colors= #ccaa88; #ddaacc; lg(#ccaa88, #ddaacc)
|charas= 东云绘名；晓山瑞希；合唱(@nolink)
[00:23.00]@2你说到底为什么 都是我的错
[00:24.00]@2都把爱情想得太美现实太诱惑
[00:25.00]
[00:30.00]@1到底为什么 让你更难过
[00:31.00]@1这样爱你除了安慰我还能怎么做
```

### 头部声明

| 声明 | 格式 | 说明 |
| ------ | ------ | ------ |
| `\|colors=` | `颜色1; 颜色2; ...` | 每个角色的文字颜色，用 `;` 分隔 |
| `\|charas=` | `名字1；名字2；...` | 每个角色的名字，用 `；` 或 `;` 分隔 |

- 颜色支持 CSS 颜色值（`#rrggbb`）
- `lg(r, g, b)` → 会被转换为 `linear-gradient(180deg, r, g, b)` 渐变
- 角色名末尾 `(@nolink)` → 标记为合唱 ID，不绑定具体角色头像
- 第一个 `@nolink` 的 ID 会被识别为 `_chorusId`，合唱时自动汇总前面所有角色

> 此处内容基于 [萌娘百科](https://zh.moegirl.org.cn/25%E7%82%B9%EF%BC%8CNightcord%E8%A7%81%E3%80%82%E6%BC%94%E5%94%B1%E6%AD%8C%E6%9B%B2) 的 WikiText 分词进一步处理

### 时间轴行

``` xlrc
[mm:ss.xx]@角色ID歌词文本
```

- `@数字` — 当前行演唱的角色 ID（对应头部声明顺序，从 1 开始）
- 同一行可多次切换角色：`@1文本A@2文本B@3文本C`
- 空内容行 `[00:14.39]` → 间奏间隙

>其实就是普通的 `lrc` ...

## ⌨️ 键盘快捷键

| 按键 | 功能 |
| ------ | ------ |
| `Space` | 播放 / 暂停 |
| `Shift + P` | 播放 / 暂停（备用） |
| `←` | 后退 5 秒 |
| `→` | 前进 5 秒 |

## 🧪 Demo 调试模式

在播放器 URL 后添加 `?demo` 即可开启调试可视化：

```text
player.html?id=19&demo
```

效果：

- 歌词滚动区域显示红色虚线边框
- 当前行高亮显示黄色边框
- 文字区域显示青色虚线边框
- 居中焦点线可见
